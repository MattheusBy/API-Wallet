## Usage
```bash
git clone https://github.com/MattheusBy/API-Wallet.git
cd API-Wallet
docker-compose build
docker-compose up
```
After that visit:

http//:0.0.0.0:80/     API-aplication 

http//:0.0.0.0:8082/ui/    SwaggerUI
